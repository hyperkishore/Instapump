InstaPump for Safari

Desktop Safari bookmarklet - click to inject InstaPump on any Instagram page

InstaPump

How to Install

  1. Drag the purple button above to your Safari bookmarks bar
  2. Go to instagram.com/reels
  3. Click the bookmarklet in your bookmarks bar
  4. InstaPump will inject and activate!
Note: If bookmarklet injection is blocked by CSP (Content Security Policy), use the Console method below instead.

Alternative: Safari Developer Console

If the bookmarklet doesn't work, paste this in Safari's Developer Console:

fetch('https://raw.githubusercontent.com/hyperkishore/Instapump/main/repo/userscript/instapump.user.js')
  .then(r => r.text())
  .then(code => {
    // Extract IIFE (skip userscript header)
    const start = code.indexOf('(function()');
    if (start === -1) throw new Error('Invalid script');
    eval(code.substring(start));
    console.log('InstaPump loaded!');
  })
  .catch(e => console.error('Failed:', e));

Enable Safari Developer Tools

  1. Open Safari Preferences (Cmd + ,)
  2. Go to Advanced tab
  3. Check "Show Develop menu in menu bar"
  4. Go to instagram.com/reels
  5. Open Console: Cmd + Option + C
  6. Paste the code above and press Enter
