Install InstaPump

Two steps. Under 2 minutes. Then you're free.

~2 minutes
1

Install the Userscripts App

A free app that lets Safari run custom scripts. Trusted by thousands, open source, no data collection.

Get on App Store (Free)

Enable Extension & Install Script

Tap download to open the script in a new tab, then follow these steps in that tab.

Download InstaPump
  • A Tap the aA button (top-left of Safari)
  • B Tap Manage Extensions → Toggle Userscripts ON
  • C Allow for All Websites, then tap Done
  • D Tap the puzzle piece icon (bottom toolbar) → Install
Why "All Websites"? The extension needs to detect when you're on Instagram. It only activates on instagram.com.

Install the Userscripts App

Enable Extension & Install Script

Click download to open the script in a new tab, then follow these steps in that tab.

Download InstaPump
  • A Open Safari → Settings → Extensions
  • B Check the box next to Userscripts
  • C Allow for All Websites (or instagram.com)
  • D Click the puzzle piece icon in toolbar → Install
