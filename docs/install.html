Install InstaPump

Three steps. Under 2 minutes. Then you're free.

~2 minutes
1

Install the Userscripts App

Userscripts is a free app that lets Safari run custom scripts. It's made by an indie developer and is completely safe.

Open in App Store
Free & Open Source. The app is trusted by thousands of users and doesn't collect any data.
2

Download InstaPump Script

Tap the button below. When prompted, choose "Save to Files" and select the Userscripts folder in iCloud Drive.

Download InstaPump
  • A Tap "Download InstaPump" above
  • B When the file opens, tap the Share button (box with arrow)
  • C Choose "Save to Files"
  • D Navigate to iCloud Drive → Userscripts folder
  • E Tap "Save"
Can't find the Userscripts folder? Open the Userscripts app first—it creates the folder automatically.
3

Enable in Safari Settings

Tell Safari to trust the Userscripts extension.

  • A Open Settings app on your iPhone/iPad
  • B Scroll down and tap Safari
  • C Tap Extensions
  • D Tap Userscripts
  • E Toggle it ON
  • F Set "Allow" to All Websites (or just instagram.com)
Why "All Websites"? Userscripts needs permission to run on Instagram. InstaPump only activates on instagram.com—it ignores everything else.
1

Install the Userscripts App

Userscripts is a free Safari extension for Mac. Get it from the Mac App Store.

Open in Mac App Store
2

Download InstaPump Script

Click below to download, then move the file to your Userscripts folder.

Download InstaPump
  • A Click the download button above
  • B Open Finder and go to your Downloads folder
  • C Move the file to: iCloud Drive → Userscripts
Tip: Open the Userscripts app first—it will show you exactly where to put scripts.
3

Enable in Safari

Allow the Userscripts extension to run.

  • A Open Safari → Settings (or Preferences)
  • B Click the Extensions tab
  • C Check the box next to Userscripts
  • D Click on Userscripts and set permissions for instagram.com
✓

You're ready!

Open Instagram Reels in Safari and start swiping.
