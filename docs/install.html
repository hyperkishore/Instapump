Install InstaPump

Three steps. Under 2 minutes. Then you're free.

~2 minutes
1

Install the Userscripts App

A free app that lets Safari run custom scripts. Trusted by thousands, open source, no data collection.

Get on App Store (Free)
2

Enable the Extension

Allow Userscripts to run in Safari. This only takes a minute.

  • A Open Settings → Safari → Extensions
  • B Tap Userscripts and toggle it ON
  • C Set permission to All Websites
Why "All Websites"? The extension needs permission to detect when you're on Instagram. It only activates on instagram.com.
3

Install InstaPump

Tap the button below, then tap the Userscripts icon in Safari's toolbar to install.

Download InstaPump

After tapping download, look for the puzzle piece icon in Safari's toolbar, tap it, then tap "Install"

Don't see the icon? Tap the aA button in Safari's address bar, then tap "Userscripts".
